# Streamlit Cloud Deployment - Quick Start Guide

## 🚀 Deploy to Streamlit Cloud in 5 Minutes

Your code is now on GitHub! Follow these steps to deploy to Streamlit Cloud and get a live URL.

---

## Step 1: Go to Streamlit Cloud

Visit: https://share.streamlit.io/

---

## Step 2: Sign In or Create Account

- Click "Sign in with GitHub"
- Authorize Streamlit to access your GitHub account
- (If you don't have a Streamlit account, it will create one automatically)

---

## Step 3: Deploy Your App

1. Click **"New app"** button
2. Fill in the form:
   - **Repository:** `Jbee-k/wim-matchmaking-platform2`
   - **Branch:** `main`
   - **Main file path:** `app.py`

3. Click **"Deploy!"**

---

## Step 4: Wait for Deployment

Streamlit will:
- Clone your repository
- Install dependencies from `requirements.txt`
- Start your app
- Give you a live URL (usually takes 2-5 minutes)

You'll see a URL like: `https://wim-matchmaking-platform2.streamlit.app`

---

## Step 5: Share the URL

Once deployed, you can:
- Share the URL with KAM and participants
- Access it from any browser
- The app will auto-update when you push new code to GitHub

---

---

## Troubleshooting:

**If deployment fails:**
1. Check that `requirements.txt` has all dependencies
2. Check that `app.py` exists in the root directory
3. Check that `.streamlit/config.toml` is correct
4. Try redeploying

**If app is slow:**
- Free tier has resource limits
- Upgrade to paid tier if needed

**If you need to update the app:**
- Just push new code to GitHub
- Streamlit will auto-redeploy
